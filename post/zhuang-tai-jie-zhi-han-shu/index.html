状态价值函数

2022-04-25

强化学习的状态价值函数与价值期望损失问题

样本平均估计价值

在多臂老虎机问题中：

Qn+1=1n∑i=1nRiQ_{n+1} = \frac{1}{n} \sum_{i = 1} ^{n} R_i Qn+1​=n1​i=1∑n​Ri​

样本平均进行价值估计

Q1Q_1Q1​表示行动的初始价值估计

QnQ_nQn​表示将进行第nnn次动作之前对动作价值的估计值

Qn+1Q_{n+1}Qn+1​表示采取了n次行动之后对动作价值的估计值

对如上公式进行推导

Qn+1=1n∑i=1nRiQ_{n+1} = \frac{1}{n} \sum_{i = 1} ^{n} R_i Qn+1​=n1​i=1∑n​Ri​

Qn+1=1n(Rn+∑i=1n−1Ri)Q_{n+1} = \frac{1}{n}( R_n + \sum_{i = 1} ^{n-1} R_i ) Qn+1​=n1​(Rn​+i=1∑n−1​Ri​)

Qn+1=1n(Rn+(n−1)Qn)Q_{n+1} = \frac{1}{n}( R_n + (n-1)Q_n ) Qn+1​=n1​(Rn​+(n−1)Qn​)

Qn+1=Qn+1n(Rn−Qn)Q_{n+1} = Q_n + \frac{1}{n}( R_n - Q_n ) Qn+1​=Qn​+n1​(Rn​−Qn​)

对新得到的公式做一个简单解释：

在从旧估计QnQ_nQn​得到新估计Qn+1Q_{n+1}Qn+1​的过程中，实际上是对旧估计的值再加上新一次实验产生误差值的加权，其中新一次误差的权重为1n\frac{1}{n}n1​。

由1n\frac{1}{n}n1​这个权重可以看出来，在不断的样本平均估计过程中，每次新的实验引入对旧估计的修正，但修正的幅度随着实验次数nnn越来越大而权重逐渐减小。这使得在从头至尾的所有实验中，每次实验数据被平等对待，达成了样本平均估计。

改进工作

实际问题中，往往状态会发生改变，价值会更迟体现，接近游戏结束的数据修正也许更加重要。观察上式：

Qn+1=Qn+1n(Rn−Qn)Q_{n+1} = Q_n + \frac{1}{n}( R_n - Q_n ) Qn+1​=Qn​+n1​(Rn​−Qn​)

其中1n\frac{1}{n}n1​被认为是学习步长，是逐渐缩减的。为了更加贴近现实问题，可以将其设置为常数α\alphaα，则有公式：

Qn+1=Qn+α(Rn−Qn)Q_{n+1} = Q_n + \alpha( R_n - Q_n ) Qn+1​=Qn​+α(Rn​−Qn​)

对其反向推导得到：

Qn+1=(1−α)nQ1+∑i=1nα(1−α)n−iRiQ_{n+1} = (1-\alpha)^n Q_1 + \sum_{i=1}^n \alpha (1-\alpha)^{n-i}R_i Qn+1​=(1−α)nQ1​+i=1∑n​α(1−α)n−iRi​

有参考过程如下

Q1=Q1Q_1 = Q_1 Q1​=Q1​

Q2=Q1+α(R1−Q1)Q_2 = Q_1 + \alpha(R_1 - Q_1) Q2​=Q1​+α(R1​−Q1​)

Q3=Q2+α(R2−Q2)Q_3 = Q_2 + \alpha(R_2 - Q_2) Q3​=Q2​+α(R2​−Q2​)

将Q2Q_2Q2​式代入Q3Q_3Q3​式中，有：

Q3=(Q1+α(R1−Q1))+α(R2−(Q1+α(R1−Q1)))Q_3 = (Q_1 + \alpha(R_1 - Q_1)) + \alpha(R_2 - (Q_1 + \alpha(R_1 - Q_1))) Q3​=(Q1​+α(R1​−Q1​))+α(R2​−(Q1​+α(R1​−Q1​)))

Q3=Q1+αR1−αQ1+αR2−αQ1−α2R1+α2Q1Q_3 = Q_1 + \alpha R_1 - \alpha Q_1 + \alpha R_2 - \alpha Q_1 - \alpha ^2 R_1 + \alpha ^2 Q_1 Q3​=Q1​+αR1​−αQ1​+αR2​−αQ1​−α2R1​+α2Q1​

Q3=(1−α)2Q1+∑i=12α(1−α)2−iRiQ_3 = (1 - \alpha) ^2 Q_1 + \sum_{i=1}^{2} \alpha(1-\alpha)^{2-i}R_i Q3​=(1−α)2Q1​+i=1∑2​α(1−α)2−iRi​
